## Installation

```bash
$ git clone git@github.com:Abbracx/test-city-ventures.git
```
After cloning the repo using either ssh or http, install project dependencies with yarn.
```bash
$ cd test-city-ventures/server
```

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev
```

## Test

```bash
# unit tests
$ yarn test
```
